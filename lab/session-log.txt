================================================================================
BUGS FIXED
================================================================================

1. MISSING tools/ DIRECTORY
   - tools/ was at /workspaces/mcpAgentLab/tools/ (root level)
   - server.js imported from ./tools/ (relative to lab/)
   - Fix: copied tools into lab/tools/ so all deps resolve from lab/node_modules

2. WRONG LOGGER IMPORT PATH
   - Tools imported from '../lib/logger.js' — lib/ didn't exist
   - Fix: updated to './logger.js' (then later '../logger.js' after logger
     was moved to lab/)

3. ZOD VERSION MISMATCH (_zod error)
   - Tools used 'import { z } from "zod"' (Zod v4 classic)
   - MCP SDK (v1.29.0) uses zod/v4-mini's toJSONSchema internally
   - z.record(z.string()) without a key type crashes toJSONSchema
   - Fix 1: switched tools to 'import { z } from "zod/v4"'
   - Fix 2: changed z.record(z.string()) to z.record(z.string(), z.string())

4. claude_desktop_config.json NOT USED BY CLAUDE CODE
   - Manually editing ~/.claude/claude_desktop_config.json has no effect
   - Fix: use 'claude mcp add' CLI command instead
   - Config is saved to /home/node/.claude.json for the project
